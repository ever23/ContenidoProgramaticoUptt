## 🤖 UPTT Planner: Automatización Inteligente
Sistema basado en **Model Context Protocol (MCP)** para la generación de cronogramas institucionales en Excel.

### Capacidades del Sistema:
1. **Generación Masiva:** Crea planillas mensuales respetando asuetos y feriados.
2. **Lógica de Cátedra:** Asegura automáticamente una clase de contenido antes de cada evaluación.
3. **Persistencia Institucional:** Los archivos se generan bajo el formato oficial de la UPTT.

### Ejemplos de Interacción:
> *"Genera las planillas de Algorítmica I-I para el trimestre de mayo. Clases los martes."*
> *"Ajusta el cronograma de Redes para mover la evaluación 3 a la semana 10."*

---

## ⚙️ Configuración del Entorno
1. Clonar repositorio e instalar dependencias: `npm install`.
2. Registrar servidor MCP en `mcp_config.json`:
```json
{
  "mcpServers": {
    "uptt-planner": {
      "command": "node",
      "args": ["C:\\Ruta\\mcp-uptt-work\\server.js"]
    }
  }
}
```
