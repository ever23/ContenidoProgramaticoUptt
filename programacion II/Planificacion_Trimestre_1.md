# Planificación Académica: Programación II (Trimestre 1)

Este trimestre se enfoca en el estudio profundo de la Programación Orientada a Objetos y el inicio del desarrollo de interfaces web.

---

## 📅 Trimestre 1: Lógica de Objetos e Interfaz (Unidades 1 a 5, 7 y 8)
**Periodo:** 26/01/2026 al 25/04/2026

### 📝 Evaluación 1: Paradigma de POO, Clases y Objetos (Unidad 1 y 2)
**Objetivo:** Comprender y aplicar los fundamentos del paradigma orientado a objetos y el modelado de sistemas.
- Paradigma de la Programación Orientada a Objetos (POO).
- Definición de Lenguaje de POO: Clases y Objetos.
- Atributos y Métodos.
- Constructores y Destructores.
- Especificadores de Acceso (Público, Privado, Protegido).
- **Actividad:** **Examen Escrito.** (25%)

### 📝 Evaluación 2: Herencia, Polimorfismo e Interfaces (Unidad 3, 4 y 5)
**Objetivo:** Implementar mecanismos avanzados de reutilización y abstracción de código.
- Herencia: Conceptos, tipos y aplicación.
- Polimorfismo: Sobrecarga y sobreescritura de métodos.
- Definición y uso de Interfaces en el desarrollo de software.
- **Actividad:** **Evaluación en Laboratorio.** (25%)

### 📝 Evaluación 3: Herramientas de Diseño de Interfaz (Unidad 7)
**Objetivo:** Dominar el lenguaje de marcado y estilizado para la creación de interfaces de usuario.
- Herramientas de Interfaz: HTML (Estructura semántica, etiquetas, formularios).
- CSS (Cascading Style Sheets): Selectores, modelo de caja, diseño responsivo.
- **Actividad:** **Examen Escrito.** (25%)

### 📝 Evaluación 4: Programación de Script - Interactividad (Unidad 8)
**Objetivo:** Aplicar lógica de programación para la manipulación dinámica de la interfaz web.
- Programación de Script: Manipulación del DOM (Document Object Model).
- Manejo de Eventos y variables en el script.
- Interactividad básica de elementos en la web.
- **Actividad:** **Evaluación en Laboratorio.** (25%)

---

## ✅ Resumen del Trimestre
Al finalizar este trimestre, el alumno domina la lógica de objetos y es capaz de construir interfaces web interactivas cumpliendo con los estándares de la industria.
