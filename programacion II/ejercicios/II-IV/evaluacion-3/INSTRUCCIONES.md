# Laboratorio 3: Maquetación Web (HTML y CSS)

## Objetivo
Aplicar los conceptos del *Capítulo I, II y III* de la Lectura 3. El objetivo es que aprendas a crear una estructura visual (HTML) y a darle estilos modernos, colores y responsividad (CSS). Dejaremos la consola atrás para adentrarnos en la Interfaz Gráfica (UI).

## Tareas del Alumno
1. Abre el archivo `index.html` dando doble clic sobre él en tu explorador de archivos. Verás la página en tu navegador web.
2. **Experimentar con CSS:**
   - Abre `css/styles.css` y cambia el color principal del encabezado (`.header-principal`) a tu color favorito (puedes usar nombres en inglés como `darkred` o códigos Hexadecimales como `#8e44ad`).
   - Agranda el tamaño de fuente (`font-size`) del botón "Guardar Estudiante".
3. **Ampliar el HTML:**
   - En `index.html`, dentro del formulario, agrega un nuevo campo (input) llamado "Edad" justo debajo de la Cédula. Asegúrate de ponerle su etiqueta `<label>`.
   - Modifica la tabla para incluir una nueva columna "Edad" y actualiza los datos "quemados" de ejemplo para que también muestren una edad.
4. Redimensiona la ventana de tu navegador para que veas cómo el diseño pasa de dos columnas a una sola (Diseño Responsivo).
