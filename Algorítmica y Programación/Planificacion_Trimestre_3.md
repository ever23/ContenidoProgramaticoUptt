# Planificación Académica: Algorítmica y Programación (Trimestre 3)

Este documento detalla la planificación del tercer trimestre, adaptando los conceptos de punteros y estructuras dinámicas al ecosistema de JavaScript.

---

## 📅 Trimestre 3: Referencias, Recursividad y Estructuras Dinámicas

### 📝 Evaluación 1: Punteros y Gestión de Memoria (Unidad 11)
**Objetivo:** Comprender cómo se gestiona la memoria en la computadora y la diferencia entre valores y referencias.
- **Teoría:** Memoria Stack (Pila) vs Heap (Montículo).
- **JavaScript:** Tipos primitivos (Valor) vs Tipos de Objeto (Referencia).
- **Práctica:** Mutabilidad de objetos y arreglos. "Simulación" de punteros mediante referencias de objetos.

### 📝 Evaluación 2: Listas Enlazadas (Unidad 12)
**Objetivo:** Construir estructuras de datos dinámicas de forma manual para entender el funcionamiento interno de las colecciones.
- El concepto de **Nodo** (Dato + Referencia al siguiente).
- Construcción de una Lista Simplemente Enlazada manual (sin usar los métodos nativos de Array).
- Operaciones: Insertar al inicio, al final y recorrer la lista.

### 📝 Evaluación 3: Recursividad (Unidad 13)
**Objetivo:** Resolver problemas complejos mediante la descomposición en subproblemas más pequeños del mismo tipo.
- Fundamentos: Caso base y Caso recursivo.
- El Stack de ejecución (Call Stack).
- Ejercicios clásicos: Factorial, Fibonacci y búsqueda recursiva.

### 📝 Evaluación 4: Estructuras Dinámicas Avanzadas (Unidad 14)
**Objetivo:** Implementar estructuras de datos con reglas específicas de acceso y salida.
- **Pilas (Stack):** Lógica LIFO (Last In, First Out).
- **Colas (Queue):** Lógica FIFO (First In, First Out).
- **Introducción a Árboles Binarios:** Concepto de raíz, ramas y hojas.
